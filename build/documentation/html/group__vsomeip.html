Detailed Description

The vsomeip module contains all elements a user applications needs to:

  • offer SOME/IP service instances
  • request SOME/IP service instances
  • offer SOME/IP eventgroups
  • subscribe to SOME/IP eventgroups
  • send and receive SOME/IP messages (request/response)
  • implement SOME/IP events and fields

◆ set_instance()

virtual VSOMEIP_EXPORT void vsomeip_v3::message_base::set_instance ( instance_t  _instance)
pure virtual

Set the instance identifier in the message header.

To address the correct service instance, vsomeip uses the instance identifier. For external services it is mapped to a IP address and port combination before the message is sent. For internal messages is transferred as additional data appended to the SOME/IP messages. Therefore, before sending a message, a user application must set the instance identifier.

Implemented in vsomeip_v3::message_base_impl.
