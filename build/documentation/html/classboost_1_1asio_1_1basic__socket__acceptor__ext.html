Detailed Description

template<typename Protocol, typename SocketAcceptorService = socket_acceptor_service_ext<Protocol>>
class boost::asio::basic_socket_acceptor_ext< Protocol, SocketAcceptorService >

Provides the ability to accept new connections.

The basic_socket_acceptor_ext class template is used for accepting new socket connections.

Thread Safety
Distinct objects: Safe.
Shared objects: Unsafe.
Example
Opening a socket acceptor with the SO_REUSEADDR option enabled:
boost::asio::ip::tcp::acceptor acceptor(io_service);
boost::asio::ip::tcp::endpoint endpoint(boost::asio::ip::tcp::v4(), port);
acceptor.open(endpoint.protocol());
acceptor.set_option(boost::asio::ip::tcp::acceptor::reuse_address(true));
acceptor.bind(endpoint);
acceptor.listen();

The basic_socket_acceptor_ext class template is used for accepting new socket connections.
