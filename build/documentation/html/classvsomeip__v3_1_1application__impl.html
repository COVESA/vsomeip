◆ init()

bool vsomeip_v3::application_impl::init ( )
virtual

Initializes the application.

The init method must be called first after creating a vsomeip application and executes the following steps to initialize it:

  • Loading the configuration from a dynamic module
    • Loading the configuration from a .json file or
    • Loading the configuration from compiled data (not yet available)
  • Determining routing configuration and initialization of the routing itself
  • Installing signal handlers
