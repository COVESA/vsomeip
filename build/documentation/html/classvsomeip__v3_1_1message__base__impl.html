◆ set_instance()

void vsomeip_v3::message_base_impl::set_instance ( instance_t  _instance)
virtual

Set the instance identifier in the message header.

To address the correct service instance, vsomeip uses the instance identifier. For external services it is mapped to a IP address and port combination before the message is sent. For internal messages is transferred as additional data appended to the SOME/IP messages. Therefore, before sending a message, a user application must set the instance identifier.
