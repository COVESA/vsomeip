Detailed Description

Representation of a DLT trace channel.

A trace channel contains one or more filters that specify the messages that are forwarded to the trace.

Add a filter to the channel.

Add a filter containing a matches range to the channel. The filter matches if the message identifiers lie within the range specified by from and to. Thus, the messages service identifier is greater equal than the service identifier specified in from and less equal than the service identifier specified in to.

Note: from and to must not contain wildcards (ANY_INSTANCE, ANY_SERVICE, ANY_METHOD).

Parameters
_fromTuples specifying the matching message with the smallest identifiers.
_fromTuples specifying the matching message with the greatest identifiers.
_is_positiveTrue for positive filters, false for negative filters. Default value is true.
Returns
Filter identifier of the added filter or FILTER_ID_ERROR if adding failed.
